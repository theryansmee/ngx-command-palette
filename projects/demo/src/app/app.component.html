ngx-command-palette demo

Press Cmd+K (or Ctrl+K) to open the command palette.

Dashboard Settings Billing Profile

Things to try

Search

  • Type dash or bill to find routes
  • Type notif or copy to find custom action commands

Async providers (prefix routing)

Prefixes are not built into the library. They are configured per-provider by the developer, so you can use whatever prefixes make sense for your app.

  • @alice or @bob to search users (simulated 400ms API delay)
  • #billing or #keyboard to search help articles (min 2 chars, 300ms delay)

Contextual commands

  • On Dashboard: "Export Dashboard Data" and "Refresh Widgets" appear
  • On Settings pages: "Reset Settings to Defaults" appears
  • Navigate away and they disappear from results
