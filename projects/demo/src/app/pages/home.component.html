ngx-command-palette

A keyboard-driven command palette for Angular. Two lines of setup and all your routes are searchable.

Press Cmd+K (or Ctrl+K) right now to try it. Every page on this site is registered automatically from the router config.

You can also try "Toggle Dark Mode", "Copy Install Command", or "Open GitHub Repository". Those are custom commands registered in the app component.

Features

  • Auto-registers routes from your Angular Router config
  • Lazy-load aware. Re-scans routes as lazy modules load
  • Async search providers with per-provider debounce and loading states
  • Prefix routing. Scope providers behind prefixes like @ or #
  • Contextual commands. Show or hide commands based on route or dynamic conditions
  • Fuzzy search with scoring for exact, prefix, word boundary, and character matches
  • Recent commands tracked in localStorage with a recency boost
  • Accessible. WAI-ARIA combobox pattern, focus trapping, screen reader announcements
  • Fully themeable via CSS custom properties
  • Signal-based reactive state with Angular signals

Quick Install

ng add @theryansmee/ngx-command-palette

That's it. See Getting Started for the full setup guide.
