Getting Started

Up and running in under a minute.

Automatic Setup

ng add @theryansmee/ngx-command-palette

This adds provideCommandPalette() to your app config, imports the component, and adds <cmd-palette /> to your root template. You're done.

Manual Setup

1. Install the package

npm install @theryansmee/ngx-command-palette
# or
yarn add @theryansmee/ngx-command-palette
# or
pnpm add @theryansmee/ngx-command-palette

2. Provide the command palette

// app.config.ts
import { ApplicationConfig } from '@angular/core';
import { provideRouter } from '@angular/router';
import { provideCommandPalette } from '@theryansmee/ngx-command-palette';
import { routes } from './app.routes';

export const appConfig: ApplicationConfig = {
  providers: [
    provideRouter(routes),
    provideCommandPalette(),
  ],
};

3. Add the component to your root template

<!-- app.component.html -->
<cmd-palette />
<router-outlet />

That's it. Press Cmd+K (or Ctrl+K) and all your routes with a title are searchable. Routes without a title are still registered with a label generated from the path.

Angular Version Support

Each Angular major version is maintained on its own branch with a matching npm tag.

Angular Library npm tag
22.x22.x.xlatest
21.x21.x.xangular21
20.x20.x.xangular20
19.x19.x.xangular19

To install a specific Angular version:

npm install @theryansmee/ngx-command-palette@angular21
